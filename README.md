# Bricks Game

A simple HTML5 Canvas starter for a brick-breaker style game (Breakout / Arkanoid).

Quick start

1. Install and run the dev server:

   ```bash
   npm start
   ```

   This runs `npx serve . -l 3000` and serves the project at http://localhost:3000

2. Open `index.html` in your browser or visit the local server address above.

Project layout

- `index.html` — main page with a canvas
- `src/game.js` — minimal game loop and canvas setup
- `src/style.css` — basic styles
- `assets/` — placeholder for images/sounds
